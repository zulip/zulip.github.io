Contribute to the open source project

We'd love your help

Zulip welcomes all forms of contributions! There's a ton of stuff to do. Here's just a small sampling of ways you can get started:

  • Write an integration for your favorite service
  • Write a docker file to make Zulip available to dockerized servers
  • Add a service monitoring dashboard
  • Add your favorite emoji

Get started

Learn how to set up a Zulip server development environment!

Zulip has a growing collection of developer documentation available on Read The Docs.

Zulip also has a development discussion mailing list. Please reach out to us if you have any questions!
